# 🏏 RCB Premium Calculator 

**Live Demo:** [View Live Project](https://sct-wd-02-seven.vercel.app/)

## 📝 About The Project
This is a fully functional web-based calculator with a custom **Royal Challengers Bengaluru (RCB)** theme. It features a premium dark glassmorphism UI with signature red and gold accents. 

This project was built as **Task 02** during my Web Development Internship at **SkillCraft Technology**.

## ✨ Key Features
* **Standard Math Operations:** Addition, subtraction, multiplication, division, and percentages.
* **Premium UI:** Glassmorphism design with a dark/gold theme and hover animations.
* **Easter Egg Logic:** Pressing the golden combo keys (`7`, `8`, `9`) in sequence bypasses the math and flashes a bold **"RCB - PLAY BOLD 👑"** victory screen!
* **Responsive Design:** Works smoothly on both desktop and mobile devices.

## 🛠️ Technologies Used
* **HTML5:** Semantic structure.
* **CSS3:** Custom styling, CSS grid, flexbox, and animations.
* **Vanilla JavaScript:** DOM manipulation, event listeners, and calculator logic.
